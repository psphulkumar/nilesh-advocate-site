# Advocate Nilesh Kumar - Professional Website

This is a professional website for Advocate Nilesh Kumar, a lawyer practicing in Begusarai District Court.

## Files Structure
- `index.html` - Home page
- `about.html` - About page
- `services.html` - Legal services page
- `contact.html` - Contact page with form
- `style.css` - Main stylesheet
- `script.js` - JavaScript for form validation
- `robots.txt` - For SEO
- `sitemap.xml` - XML sitemap for search engines
- `images/` - Directory for images (add your photo here)

## Setup Instructions

1. **Add Your Photo**: Replace `images/nilesh-kumar.jpg` with your actual photo. The image should be square or circular crop-friendly.

2. **Update Contact Information**: Edit the contact details in all HTML files:
   - Replace `[Your Phone Number]` with your actual phone number
   - Replace `[Your Email Address]` with your email

3. **Domain Setup**: 
   - Update `robots.txt` and `sitemap.xml` with your actual domain name
   - Replace `https://yourdomain.com` with your website URL

4. **Form Handling**: The contact form currently shows an alert. For production, integrate with a backend service like:
   - Formspree
   - Netlify Forms
   - PHP backend
   - Or any email service
